#Заперечення

<center><h3>Форма</h3></center>

| <h4>-</h4> |Subject | to be + not| Verb + -ing | Object/Adverbial modifier |
| -- | -- | -- | -- | -- | 
| | I | am not | working | now. |
| | He | is not |  playing | computer games at the moment.|
| | We | are not | preparing | for exams. |
<br>
